### 4. [Final Project: Machine Learning in Finance](./Final%20Project)
**Title:** Hierarchical Ensemble System for SPY Prediction
* **Overview:** A comprehensive study applying advanced machine learning techniques to predict S&P 500 ETF (SPY) movements.
* **Methodology:**
    * **Feature Engineering:** Implemented **Fractional Differentiation** (d=0.4) to preserve memory while achieving stationarity.
    * **Model Architecture:** Constructed a **Two-Layer Blending Ensemble** classifier integrating Logistic Regression, Random Forest, LightGBM, XGBoost, and Gradient Boosting.
    * **Unsupervised Learning:** Utilized **Self-Organizing Maps (SOM)** for feature dimensionality reduction and clustering.
* **Key Findings:**
    * Achieved a **Sharpe Ratio of 1.43** in out-of-sample testing.
    * Post-trade attribution analysis revealed that performance was primarily driven by **Regime-Based Risk Overlays (Trend Filtering)** rather than raw predictive accuracy.
* **File:** `ML LIU JingZhong REPORT.pdf`
